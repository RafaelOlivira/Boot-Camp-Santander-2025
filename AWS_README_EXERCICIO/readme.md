## Introdução
Este relatorio apresenta o processo de implementação de ferramentas na empresa Farma Company, realizado por Rafael Oliveira. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos especÃ­ficos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- Amazon EC2 com Auto Scaling
- Otimização de custos de infraestrutura computacional
- Utilização de instâncias EC2 configuradas com Auto Scaling para ajustar automaticamente a capacidade de processamento conforme a demanda. Essa    abordagem evita o pagamento por recursos ociosos, garantindo economia imediata e melhor aproveitamento da infraestrutura.

Etapa 2: 
- Amazon S3 (Simple Storage Service)
- Redução de custos com armazenamento de dados
- Migração de arquivos locais e dados pouco acessados para o Amazon S3, utilizando políticas de ciclo de vida para mover objetos para classes de armazenamento mais econômicas, como S3 Standard-IA e S3 Glacier, reduzindo significativamente os custos de armazenamento.

Etapa 3: 
- AWS Lambda
- Execução de código sob demanda sem necessidade de servidores
- Implementação de funções Lambda para tarefas automatizadas e processos event-driven, eliminando a necessidade de servidores dedicados e reduzindo custos operacionais, uma vez que o pagamento ocorre apenas pelo tempo de execução do código.



## Conclusão
A implementação de ferramentas na empresa Farma Company tem como esperado a redução de custos operacionais, maior escalabilidade, melhor gerenciamento de recursos e aumento da eficiência dos processos,, o que aumentarão a eficiencia e a produtividade da empresa. Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
